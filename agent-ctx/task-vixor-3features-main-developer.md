### Summary

1. Real TradingView Chart Integration
2. Price Alert System
3. Daily Trade Signals / Recommendations

### Key Implementation Details

1. **TradingView Integration**: Uses ref-based approach to dynamically load the TradingView widget script. Supports symbol mapping (BTC/USDT → BINANCE:BTCUSDT). Handles cleanup on unmount. Symbol change re-initializes widget.

2. **Price Fetcher**: Multi-source price fetching with fallbacks:
   - Crypto: Binance API (price + 24h change)
   - Forex: Finnhub or exchange-rate-api
   - Gold: Finnhub commodity quotes
   - Fallback: Deterministic prices based on known base prices

3. **Alert System**: Full CRUD with server functions. Alert checker can be called via API endpoint for cron scheduling. Supports conditions: above, below, crosses_up, crosses_down. Triggers create Supabase notifications and Telegram messages.

4. **Daily Signals**: Generated by running local SMC/ICT analysis engine with real Binance OHLCV data when available. Filtered by user strategy preferences. Signal cards show entry/SL/TP with "Set Alert" action.

5. **Route Tree**: Regenerated via `npx @tanstack/router-cli generate` to include the /signals route.
